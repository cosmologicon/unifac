Robot Underground: About

Play Robot Underground

Requirements

Robot Underground requires a modern (as of 2013) web browser with JavaScript and webGL enabled.

Release History

Robot Underground was originally created for PyWeek #6, the one-week Python game programming challenge. The original entry (version 1.0.1) was submitted to the competition on 06 Apr 2008 by Team Super Effective:

  • Adam Biltcliffe (adam)
  • Martin O'Leary (Martin)
  • Carrie Oliver (Carrie)
  • Richard Thomas (Chard)
  • John-Joseph Wilks (Garath)

Robot Underground version 1.0.4 was published by Super Effective on 27 Apr 2008.

Christopher Night (Universe Factory games) ported version 1.0.4 to JavaScript. This became js version 0, published on 25 Nov 2013.
